Setting up your Rails app

  1. Install the Hoptoad notifier: 
    script/plugin install git://github.com/thoughtbot/hoptoad_notifier.git
  2. Put this in your config/initializers/hoptoad.rb (or download it here): 
    HoptoadNotifier.configure do |config|
  config.api_key = '5afafe618c6dbc5db23e7e3ff818b40a'
end
  3. Remove your existing Exception Notifier from your ApplicationController: 
      class ApplicationController < ActionController::Base
     include ExceptionNotifiable 
  4. Test that the notifier was correctly configured by running the included rake task: 
    rake hoptoad:test
  5. Look for the exception to appear in your errors list.
